Los Hermanos Ayar - Juego 2D en Unity

Descripción del Proyecto

Los Hermanos Ayar** es un videojuego 2D desarrollado en Unity inspirado en la leyenda peruana de los Hermanos Ayar, considerada una de las historias fundacionales del Imperio Inca. El proyecto busca combinar entretenimiento y aprendizaje mediante una experiencia interactiva que permite a los jugadores recorrer distintos escenarios inspirados en la cultura andina.

El juego ha sido diseñado para dispositivos móviles, incorporando controles táctiles mediante botones en pantalla y personajes animados utilizando Spine para lograr movimientos fluidos y atractivos.

Instalación

Requisitos

* Unity 2021 o superior
* Spine Runtime para Unity 3.8
* Git

### Pasos

1. Clonar el repositorio:

```bash
git clone https://github.com/usuario/repositorio.git
```

2. Abrir el proyecto en Unity.

3. Importar el paquete Spine Runtime:

```text
spine-unity-3.8-2021-11-10.unitypackage
```

4. Esperar a que Unity compile los recursos.

5. Ejecutar la escena principal.


Controles

Dispositivos móviles

* Botón Izquierda → Mover personaje a la izquierda.
* Botón Derecha → Mover personaje a la derecha.
* Botón Salto → Saltar obstáculos.
* Botón Acción → Interactuar con elementos del entorno.
